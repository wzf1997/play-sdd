ScratchCard 插件架构总览

遵循现有 Grid9 / BlindBox 插件模式

Web Layer — apps/web/src/games/ScratchCard/
ScratchCardGame.tsx
主组件，状态机协调者
ScratchCanvas.tsx
Canvas 刮涂组件，独立封装
WinModal.tsx
中奖弹窗 + 24h 倒计时
useScratchCardGame.ts
状态机 + BFF 调用
useCardPhysics.ts
rAF 物理引擎，每帧更新卡片位置
index.ts
插件注册 (registerGame)
↕ /api/game/scratch-card/init · play · result
BFF Layer — apps/bff/src/routes/games/scratch-card/
init.ts
生成 session，从 service 取奖品配置
play.ts
校验选中卡片，返回奖品
result.ts
查询最终结果 + 倒计时状态
↕ HTTP
Service Layer — apps/service/src/games/scratch-card/
scratch-card.ts
业务逻辑：奖品池、防重刷
SQLite
复用 campaigns / prizes 表，新增 scratch_sessions
